À propos de cette extension
---------------------------

Le but de cette extension est de rendre n'importe quel objet TypoScript défini dans le gabarit du site disponible en
tant qu'élément de contenu normal. Si vous avez, par exemple, défini un menu dans votre gabarit TS, cette extension vous
permettra de le placer dans une page, comme un élément texte ou image standard.
